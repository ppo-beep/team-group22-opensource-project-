# Food recommendation program 
이 프로젝트는 사용자의 입력 문장을 기반으로 요청 조건에 맞는 음식 메뉴를 추천합니다.

---

## 사용 패키지 및 버전

- Python 3.8 이상 권장
- Transformers (Huggingface tokenizer 사용)

> 아래 명령어를 통해 필요한 패키지를 설치할 수 있습니다:

```bash
pip install -r requirements.txt
```

## 필수 데이터 파일: menu_db.csv

- 메뉴 데이터베이스
  - menu_db.csv 파일은 각각의 음식이 어떤 키워드 속성을 가지고 있는지 저장되어 있습니다.
  - 프로그램은 menu_db.csv 파일을 읽어 JSON처럼 다루며, 각각의 음식이 어떤 키워드 속성을 가지고 있는지 비교가능한 상태로 저장됩니다.

## 실행 구조

- 조건 필터링

    - 사용자가 입력한 키워드가 음식 항목의 속성과 일치하는지 확인하고 매칭됩니다.

- 추천된 메뉴를 출력합니다.

- 반복 입력 처리
    - "종료" 혹은 "exit" 입력 시 프로그램이 종료됩니다.
  

- 실행 결과 예시

```
  간단한 음식 추천 시스템입니다.

 고를 수 있는 음식 스타일:
  - 종류: 한식, 중식, 일식, 양식
  - 온도: 따뜻한, 차가운
  - 맵기: 매콤한, 순한
  - 주재료: 밥, 면
예시: '따뜻한 면 요리 중식 먹고 싶어'
종료하려면 '종료'를 입력하세요.

먹고 싶은 음식 스타일을 말해보세요: 매운 면요리, 중식
입력 토큰: ['매', '##운', '면', '##요', '##리', ',', '중', '##식']

추천 메뉴:
  - 짬뽕 (중식 / 따뜻한 / 매콤한 / 면)
먹고 싶은 음식 스타일을 말해보세요: 차가운 음식
입력 토큰: ['차', '##가', '##운', '음', '##식']

추천 메뉴:
  - 비빔밥 (한식 / 차가운 / 매콤한 / 밥)
  - 냉면 (한식 / 차가운 / 순한 / 면)
  - 중국냉면 (중식 / 차가운 / 순한 / 면)
  - 스시 (일식 / 차가운 / 순한 / 밥)
  - 자루소바 (일식 / 차가운 / 순한 / 면)
  - 샐러드 (양식 / 차가운 / 순한 / 밥)
먹고 싶은 음식 스타일을 말해보세요: 따뜻한 밥이 먹고 싶어. 매콤했으면 좋겠어
입력 토큰: ['따', '##뜻', '##한', '밥', '##이', '먹', '##고', '싶', '##어', '.', '매', '##콤', '##했', '##으면', '좋', '##겠', '##어']
조건에 맞는 메뉴를 찾을 수 없습니다.😢
먹고 싶은 음식 스타일을 말해보세요: 종료
시스템을 종료합니다. 안녕히 가세요!

```
